## Instrucciones de Uso de la Aplicación
1. Abrir la aplicación
2. Presionar el botón "Explorar ahora"
3. En la parte inferior se podrá un obvervar un menú de navegación en la que se observarán 5 pestañas (fragments).
   - En el icono del lápiz se observará una pestaña con el título "Elementos de Texto", en ella se podrá visualizar una breve descripción de lo que son estos elementos y unops ejemplos interactivos, donde tendrá que introducir información en los cuadros de texto y depsués presionar un botón para "mandar" la información.
   - En el icono de la flecha se podrá oobservar la pestaña de "Botones", en ella se encuentra información sobre lo que son estos elementos y un ejemplos interactivos, donde se podrá poner a prueba los distintos botones que hay disponibles.
   - En el icono con dos cuadros se encuentra la pestaña de "Elementos de Selección", aquí se podrá ver información sobre lo que son estos elementos y unos ejemplos interactivos donde el usuario podrá seleccionas o no algunas opciones de distintas maneras.
   - En el icono con tres rayas se tiene la pestaña de "Listas", en ella se podrá visualizar información de estos elementos y un ejemplo de una lista.
   - Por último, en el icono de información se observará la pestaña "Elementos de Información", donde se podrá ver una descripción de este tipo de elementos y un ejemplo interactivo, donde el susario deberá accionar el botón para que vea como el texto cambia, informando que la carga se ha hecho.
